# Text

> **하나 이상의 읽기 전용 텍스트를 표시하는 뷰 (UILabel)**
> 

### modifier

- **`.lineLimit` : 줄 개수를 제한**
- **`.lineSpacing` : Text의 줄 간격**
- **`.font` : Text의 폰트**
- **`.foregroundColor` : Text의 색상을 변경**

[Text | Apple Developer Documentation](https://developer.apple.com/documentation/swiftui/text)

```swift
Text("Hamlet")
```
